## About

I build small tools, web utilities, automation scripts, and backend-focused experiments. My current direction is practical software development with a growing focus on cybersecurity, OSINT, and cleaner public project presentation.

I care about code that is direct, usable, and easy to understand. The goal is not to look busy; the goal is to ship useful work and keep improving it.
